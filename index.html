ESP32 Sterilization Monitor

Medical-grade embedded system demonstrating real-time control, safety interlocks, and deterministic state machine design.

Overview

A sterilization cycle controller built on dual ESP32 microcontrollers, implementing production-style safety protocols, FreeRTOS task management, and clean separation between control logic and user interface—principles fundamental to certified medical devices.

Dual-MCU Architecture

Separate control and UI subsystems for fault isolation and reliability.

FreeRTOS

Real-time task scheduling with priority-based execution for safety-critical operations.

Safety Interlocks

Hardware-enforced door monitoring with immediate shutdown capability.

Environmental Sensing

BME280 I²C sensor for temperature and humidity validation.

State Machine

The controller implements a deterministic finite state machine with explicit transitions and comprehensive fault handling.

Technical Implementation

Controller ESP32-S3 handling state machine, sensor I²C, relay control, and safety monitoring
UI Module Dedicated ESP32 display receiving UART telemetry for real-time visualization
Sensors BME280 temperature/humidity sensor with health validation
Safety Normally-closed door interlock, watchdog timers, defensive state logic
Firmware ESP-IDF, FreeRTOS, static memory allocation, modular task architecture

Key Features

Real-time embedded design with FreeRTOS task management and priority scheduling. Deterministic state machine implementation with explicit transitions and defensive programming. Environmental sensor integration over I²C with validation and fault detection. UART-based multi-MCU communication for control/UI separation. Hardware-enforced safety interlocks with immediate shutdown capability.

The system demonstrates production-grade embedded design principles applicable to medical devices, industrial control systems, and other safety-critical applications.
